# 機器分割機能

## 1. 目的と概要

CADとT-BOMのデータを紐づける際、特定のリストタイプの機器は**「必ず1台対1台」**でしか紐づかない仕様があります。

CAD側では常に1台ずつ登録されている一方、T-BOM側で「数量2以上」でまとめて登録されている場合、紐づけができません。そのため、**CADデータを取り込む前に**、T-BOM側の機器を1台ずつに分ける処理が「機器分割作業」です。

## 2. 実施タイミング

**CADデータとT-BOMを紐づける前**（CADWorxP&IDデータを取り込む前）に行います。  
「紐づけ作業」や「整合チェック」の事前準備段階で、分割の要否を判断します。

## 3. 対象条件

- **リストタイプ**: L121、L122、L123、L124、L128、L151、L152、L153、L154、L158、L162、L165、L841（計13種類）
- **数量**: T-BOM上で数量が**2以上**で登録されているもの
- **例外**: 予備のために計上してある機器は分割不要

## 4. 作業前の注意事項（厳守事項）

作業にあたっては以下のルールを厳守してください。

- **識別IDの保持**: T-BOMに原価計上時に付与される「識別ID」はCADとの紐づけキーです。分割時も**絶対に変更しない**こと。
- **他使用者の確認**: 「リスト一覧」画面で、他の使用者が編集していないことを確認する。
- **バックアップの取得**: 分割前の設計（機器仕様）を後から確認できるよう、作業前に必ず「機器リスト」のバックアップを取得する。
- **金額・数量の確認**: 分割の前後で「金額」と「機器数量」に差異が発生していないかを必ず確認する。

**【リスク】** 設計担当者が手動で原価を調整して登録していた場合、分割処理によってシステム本来の計算金額（分割前の単価×数量）に上書きされ、意図しない金額の差異が発生するおそれがあります。

## 5. 作業手順

1. T-BOMの表示を「**機器別表示**」に切り替える。
2. 分割対象の機器フォルダ（数量2以上の対象リストタイプ）を選択し、「**リスト編集**」をクリック。
3. 「**金額付き出力**」をクリックし、機器リストをExcelで出力。バックアップ用に作業フォルダへ保存。
4. 「リスト編集」タブ内の「**数量分に分割**」をクリック。
5. 「**すべて機器を対象にする**」にチェックを入れ、「OK」で実行。

## 6. 自動分割エラーとなるケース（手動修正）

以下に当てはまる場合はシステムが自動分割できないため、**手動で機器番号を変更**する必要があります。

- **数量の不一致**: 機器番号末尾のアルファベット数と登録数量が一致していない（例：`1P-K10421A~D` なのに数量が「3」）
- **区切り記号の欠如**: 機器番号に `,` `/` `~` `～`（全角）のいずれも含まれていない（例：`EP-5201A-C` はハイフンのみで区切りがない）
- **末尾の文字種**: 機器番号の末尾がアルファベット以外（例：`EP-5131A,B,C-1`、`1P-K10821E1~D1`）

## 7. 自動分割のロジック（参考）

マニュアルには内部プログラムは記載されていませんが、エラー条件から以下のロジックが読み取れます。このロジックに合致するデータのみ自動処理され、外れたデータは手動対応が必要です。

### ロジックの要件

- **区切り記号**: 機器番号内に `,` `/` `~` `～` のいずれかを含み、複数台のまとまりと認識されること
- **末尾のアルファベット**: 機器番号末尾がアルファベットであること（連番 A, B, C… を生成する前提）
- **数量の一致**: 機器番号から導出される台数と、T-BOM登録数量が完全に一致していること

### まとめ

**「機器番号が特定の記号で区切られており、末尾がアルファベットで終わり、そのアルファベットが示す台数とT-BOMの登録数量が完全に一致している」** という条件を満たすデータのみ自動分割可能です。
