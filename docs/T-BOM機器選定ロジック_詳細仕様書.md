## 1. 概要

### 1.1 目的

CADWorx P&IDのデータをT-BOMに取り込んだ際に、P&ID内に記載された情報からT-BOMに登録された機器を自動で選定するロジックを定義する。また、T-BOMで選定された機器・弁類の数量・寸法等から、スキッド面積・スキッド寸法を推算するロジックを定義する。

最終目的は、CADWorxフローシートに記載された仕様から精度の高い原価を短時間で作成すること（生産品質向上と生産効率向上）である。

### 1.2 適用範囲

- 機器の選定ロジックは「添付2. 対象リストタイプ」に示すリストタイプの機器を対象に、CADWorx P&IDで作図されたフローシートをT-BOMに取り込んだ際に適用される
- スキッド寸法推算ロジックは、T-BOMでの機器選定完了後に適用される
- システムの利用は任意
- 対象範囲: 国内電子・水処理の機械設計手配機器（土壌、電力、海外を除く）

## 2. 自動選定の基本思想

### 2.1 ロジックの基本構造

| 項目             | 内容                                                      |
| ---------------- | --------------------------------------------------------- |
| 選定項目の表現   | 各機器の選定項目（形式等）は「条件1、条件2、…」として定義 |
| 条件の組み合わせ | ロジックは **条件1 AND 条件2 AND …** の形式               |
| 条件の記述方法   | True or False で記載（完全一致 or 部分一致）              |

### 2.2 ロジック読み込みルール

1. ロジック1から**順番に**読み込む
2. 選定項目（候補）が**1つになった地点で終了**
3. 全ロジック読み込み後も選定項目が複数ある場合は**自動選定されない**
4. 候補が0件になった場合も**選定されない**（未積算）

### 2.3 条件の2種類の適用方式

| 方式                    | 条件番号         | 動作                                            |
| ----------------------- | ---------------- | ----------------------------------------------- |
| **AND条件**（個別条件） | 条件011〜条件165 | 候補が0件になった場合、選定されない             |
| **OR条件**（共通条件）  | 条件191〜条件197 | 番号の若い順に絞り込みを行い、1機種になれば終了 |

### 2.4 フォローしていない選定基準

本ロジックでフォローしていない選定基準は以下の通り（ただしロジック自体は本書で定義）:

1. 配管材料仕様書に記載の機器
2. メーカーのエンジニアリングが必要な機器 → 型式から選定とする

## 3. 機器自動選定ロジック

### 3.1 選定の流れ

```
CADWorx P&ID フローシート
    ↓ 取り込み
T-BOM（機器ラベル情報を抽出）
    ↓
【ステップ1】個別条件による絞り込み（AND条件）
  条件011〜条件165を順次適用
  → 各条件で候補が0件なら「未積算」で終了
  → 候補が1件になれば選定完了
    ↓
【ステップ2】共通条件による最終選定（OR条件）
  条件191 → 条件192 → … → 条件197の順に適用
  → 1機種に絞られた時点で選定完了
    ↓
選定結果 or 未積算
```

### 3.2 注意事項

1. フローシートの記述に不足がある場合や、エンジニアリングフローシート作成基準(C020-D01)から外れた描き方の場合、機器が選定されない場合や適正でない機器が選定される場合がある
2. 条件を元に絞り込んだ結果、T-BOMに該当機器が登録されていなかった場合は未積算となる。その場合はT-BOMの単価登録依頼を実施すること
3. 機器選定のロジックは変更可能。ロジックの不具合・変更の際はDX部へ相談

### 3.3 引用

- 機器ラベル: C020-D01-6 rev.4

### 6.2 工業計器の選定ロジック

工業計器（L831系）は、機械系機器とは異なる専用の選定フローを持つ。

#### 選定フロー

```
CADWorxのTAG情報
    ↓
【前処理】TAG頭文字の数字を自動削除（例: 7LIA-1111 → LIA-1111）
    ↓
【条件801】TAGパターンでリストタイプ振分（→ 添付5参照）
    ↓
【条件802】測定方式(形式) → 計器名称の照合【先頭から部分一致】
    ↓
【条件803/808】流体記号→接液適合材質で要部材質を照合【末尾から部分一致】
  ※一般計器: 条件803（適合材質1〜5位）
  ※調節弁: 条件808（本体適合材質1〜2位）
  ※不一致時は未積算
    ↓
【条件804】接続サイズ → 接続口径【先頭から部分一致】
    ↓
【条件805】接続規格【先頭から部分一致】（将来開発）
    ↓
【条件806】目盛上限値 → 入力信号測定範囲【直近上位】
    ↓
【条件807】クラス記号 → 型式選定（将来開発）
    ↓
【条件809〜815】電気機器リスト属性の照合
  ・電源種別、防爆、塩害対策、通信機能、変換器種別、洗浄仕様
    ↓
【条件810】クラス記号 → 脱脂・禁油設定
    ↓
【条件191】ベンダーリスト
    ↓
【条件196】最安値/最高値の選択
    ↓
【条件197】セグメント/グレードの選択
```

#### L831-001 流量計の条件807の特殊ロジック

流量計の場合、条件807にメーカー・形式固有の分岐ロジックがある:

- メーカーが「東京計装」かつ測定方式が「超音波式」の場合のみ条件が適用
- クラスが「1G, 1W, 1L, 1J, 1E, 1Z」で始まる → **UL350** を選定
- それ以外 → **UL330** を選定

### 9.4 スキッド寸法推算式

推算されたスキッド面積から、スキッド種類・面積に応じた片辺B(mm)を設定し、もう一方の辺を算出する。

```
スキッド辺A = スキッド面積 ÷ (スキッド片辺B ÷ 1000)
```

片辺Bは車両運送を考慮し2,200mmを最大とする。
